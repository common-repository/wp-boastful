WP Boastful integrates the best tweetback library 'boastful' by Zach Holman to your blog.

== Description ==

When I saw 'boastful' library by Zach Holman, I wanted it on my blog, so I created this plugin to easily integrate this library
to any wordpress blog. What boastful does is that it shows 'tweetback' to you blog post.

It shows you the avatar of people talking about your blog post on twitter, and the actual content of the tweet on hover of the image.

This plugin is brought to you by [WPoets](http://wpoets.com/ "WordPress Experts") team.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `wp-boastful` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. And you are done.
